安裝步驟指南

1
下載安裝檔

點擊上方按鈕下載安裝檔。
若瀏覽器阻擋下載，請點選「更多」選擇「保留」。

2
執行安裝程式

進入下載資料夾，雙擊 Setup.exe 並依照指示完成安裝。
若出現 Windows 安全提示，請點選「其他資訊」並選擇「仍要執行」。

3
登入並開始使用

安裝完成後，程式將自動啟動。請使用您目前的帳號密碼登入即可開始使用。

系統需求

⚙️
作業系統

Windows 8.1 以上

⚙️
網際網路

登入、存檔、下載時需要使用網路連線（初次使用必須登入）

⚙️
中央處理器 (CPU)

最低需求：Intel Core i3 第 6 代 / AMD Ryzen 3 以上
建議規格：Intel Core i5 第 8 代 / AMD Ryzen 5 以上

⚙️
隨機存取記憶體 (RAM)

最低需求：4GB RAM
建議規格：8GB RAM 或更高

⚙️
儲存空間

安裝所需空間：2GB
